ChatGPT Clone (Streamlit Version)
This project is a lightweight, responsive ChatGPT Clone built using the Streamlit framework and the OpenAI API. It provides a clean, user-friendly interface for interacting with Large Language Models (LLMs) without the overhead of a complex backend.

🌟 Overview
The application is designed for users who want a private, "bring-your-own-key" (BYOK) chat environment. By leveraging Streamlit's reactive capabilities, the interface mimics the standard ChatGPT experience while remaining easy to deploy and customize.

✨ Key Features
Secure API Integration: Features a dedicated sidebar for users to input their personal OpenAI API Key, ensuring no hard-coded credentials and enhanced privacy.

Minimalist UI: A clean, distraction-free chat interface titled "克隆ChatGPT" (ChatGPT Clone), featuring a central chat area and an intuitive input field.

Real-time Interaction: Built with Python-based Streamlit, allowing for fast response rendering and session state management.

Quick Deployment: Optimized for platforms like Streamlit Cloud, making it accessible via a custom .streamlit.app URL as seen in the browser address bar.

🛠️ Tech Stack
Frontend/Backend: Streamlit (Python)

Model Provider: OpenAI API

UI Components: Custom CSS and Streamlit Chat Elements

🚀 How to Use
Access the Web App: Navigate to the hosted Streamlit URL：https://fwscutmm.streamlit.app/

Authentication: Enter your valid OpenAI API Key in the sidebar input field labeled "Please enter OpenAI API Key" (请输入OpenAI API Key).

Start Chatting: Type your prompt into the bottom text box labeled "Your message" and hit Enter to receive an AI-generated response.
